### 💻 本地调试

拉取代码并切换到项目目录下：

```bash
$ git clone https://github.com/muxi-Infra/muxi-react-utils.git
$ cd PenPal
```

安装依赖：

```bash
$ bun install
$ bun run dev
```

开始调试你的调试之旅！

```bash
$ bun run dev
```
